# ⌨️ TypeRacer - Typing Speed Tester

A sleek, real-time typing speed tester built with vanilla HTML, CSS, and JavaScript.  
Test your WPM (Words Per Minute) and accuracy while typing motivational quotes.

## 🚀 Live Demo
[Click here to try it live!](https://your-username.github.io/typing-speed-tester)

## ✨ Features
- **Real-time feedback** – Highlights correct (green) and incorrect (red) characters instantly.
- **Live stats** – Tracks Time, WPM, Accuracy, and Progress as you type.
- **Dark mode** – Easy on the eyes, built with GitHub's dark theme.
- **Random quotes** – A curated list of motivational quotes to keep you inspired.
- **Responsive** – Works perfectly on desktop, tablet, and mobile.

## 🛠️ Built With
- HTML5
- CSS3 (Flexbox, Grid, custom properties)
- Vanilla JavaScript (DOM manipulation, event handling, timers)
